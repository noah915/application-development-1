Simple e-commerce (products & orders)

What the system does
--------------------
A lightweight e-commerce API that manages products, customer accounts, shopping carts, and orders. It supports product catalog retrieval, cart management, order creation and tracking, and basic inventory checks.

Who uses the API
-----------------
- Frontend web or mobile clients (shop UI)
- Internal admin dashboard for product management and order fulfillment
- Third-party services (analytics, inventory sync)

Core resources
--------------
- products
- customers
- orders
- carts
- reviews
